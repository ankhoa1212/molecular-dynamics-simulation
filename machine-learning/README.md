# Quick start

Prerequisites: ensure the `yolov5` folder is in this folder and `uv` is installed.

1. Install `uv` (example):
    ```
    pip install uv
    ```
2. Place the `yolov5` folder into this directory.
3. Sync the project:
    ```
    uv sync
    ```
4. Run the main script:
    ```
    uv run main.py
    ```

Note: activate your virtual environment if applicable.
